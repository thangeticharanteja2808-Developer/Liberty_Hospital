## 🚀 Deployment

### Deploy on Vercel (Recommended)

```bash
# Install Vercel CLI
npm install -g vercel

# Navigate to project folder
cd "Liberty Hospital"

# Deploy
vercel
```

Follow prompts → site goes live at `https://liberty-hospital.vercel.app`

### Deploy on Netlify (Alternative)

1. Go to [netlify.com/drop](https://app.netlify.com/drop)
2. Drag the **Liberty Hospital** folder into the page
3. Done in 10 seconds ✅
